# EMS 3.0 - Employee Management System

EMS 3.0 is a simple Employee Management System web application. It allows users to log in and manage both employee and job details, including adding, viewing, editing, and deleting employee information and job listings.

## Table of Contents

- [Features](#features)
- [Getting Started](#getting-started)
- [Usage](#usage)
- [Dependencies](#dependencies)
- [Contributing](#contributing)
- [License](#license)
- [Live Demo](#live-demo)

## Features

- User authentication with a login page.
- Employee Dashboard for managing employee details.
- Jobs Dashboard for managing job listings.
- Modal dialogs for adding, viewing, editing, and deleting employee information and job listings.
- Responsive design using Bootstrap.

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory:**

   ```bash
   cd EMS-3.0
   ```

3. **Install Dependencies:**

   This project uses Bootstrap, and the necessary dependencies are included via CDN. No additional installation is required.

4. **Set up a Backend:**

   Connect this frontend with a backend server to handle authentication, employee data, and job data.

5. **Open the `index.html` file:**

   Open the `index.html` file in your web browser to access the login page.

## Usage

1. **Login:**

   Access the login page (`index.html`), enter your username and password, and click "Login to continue."

2. **Employee Dashboard:**

   After successful login, navigate to the employee dashboard (`dash.html`). Here you can perform various actions, such as adding, viewing, editing, and deleting employee details.

3. **Jobs Dashboard:**

   Access the jobs dashboard (`jobs-dashboard.html`) to manage job listings. Similar to the employee dashboard, you can add, view, edit, and delete job details.

4. **Modals:**

   - **Add Employee:** Click on "Add Employee" in the employee dashboard sidebar to open the modal for adding a new employee.
   - **View Details, Edit, Delete Employee:** Click on the respective buttons in the employee list to view details, edit, or delete an employee.
   - **Add Job, View Details, Edit, Delete Job:** Similar actions can be performed in the jobs dashboard.

## Dependencies

- [Bootstrap](https://getbootstrap.com/): For styling and responsive design.
